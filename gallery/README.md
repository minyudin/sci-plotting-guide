# 图片素材库（参考用）

从网络搜集的论文插图示例，仅供个人学习和绘图风格参考。图片版权归原作者/出版方所有，请勿直接用于发表。

- `框架图-示意图加实物/` — 示意图 + 实物照片结合的实验装置/流程框架图
- `LLM框架图-第一批/`、`LLM框架图-第二批/` — 大模型相关框架图（架构、Agent、RAG、训练流程等）；`arxiv高清/` 子目录为 arXiv 论文原版高清图（文件名前缀为论文编号，如 `2312.10997_` 对应 arXiv:2312.10997）
- `数据分析图/` — SCI 论文风格统计/可视化图，文件已按图型命名分类：
  - 火山图(35)、热图(19)、聚类热图(12)、相关性热图(16)、小提琴图(36)、箱线图(34)、散点PCA图(8)、折线图(2)、气泡图(1)、多面板组合图(119)

各文件命名为「图型-序号.jpg」；框架图目录内文件为「装置示意图与实物结合-序号.jpg」「LLM框架图-序号.jpg」，arXiv 高清图前缀为论文主题+编号。
