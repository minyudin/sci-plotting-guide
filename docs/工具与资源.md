# 工具与学习资源清单

以下链接均已验证有效（2026-07）。

## Python

| 资源 | 说明 |
|---|---|
| [SciencePlots](https://github.com/garrettj403/SciencePlots) | 一行代码套用 Nature/IEEE/Science 风格的 Matplotlib 样式包（`plt.style.use('science')`） |
| [proplot](https://github.com/proplot-dev/proplot) | 面向科研出版图的 Matplotlib 高级封装 |
| [colorcet](https://github.com/holoviz/colorcet) | 感知均匀色图集合（CET 系列） |
| [Matplotlib 色图指南](https://matplotlib.org/stable/users/explain/colors/colormaps.html) | 官方色图选择说明（为什么不要用 jet） |
| [Seaborn 配色教程](https://seaborn.pydata.org/tutorial/color_palettes.html) | 定性/连续/发散配色的系统讲解 |
| [matplotlib_for_papers](https://github.com/jbmouret/matplotlib_for_papers) | 用 Matplotlib 制作出版级图表的实操教程 |
| [Scientific Visualization: Python + Matplotlib](https://github.com/rougier/scientific-visualization-book) | Nicolas Rougier 的开源科学可视化书（PDF 免费） |

## R

| 资源 | 说明 |
|---|---|
| [ggsci](https://nanx.me/ggsci/)（[GitHub](https://github.com/nanxstats/ggsci)） | NPG/AAAS/NEJM/Lancet/JAMA 等期刊配色的 ggplot2 标度 |

## 在线配色工具

| 工具 | 用途 |
|---|---|
| [ColorBrewer 2.0](https://colorbrewer2.org/) | 经典科研/地图配色生成器，可筛选色盲安全 |
| [Paul Tol 配色](https://sronpersonalpages.nl/~pault/) | 色盲友好定性/连续配色的权威参考 |
| [Fabio Crameri 科学色图](https://www.fabiocrameri.ch/colourmaps/) | 感知均匀科学色图（batlow 等），提供各软件格式下载 |
| [Coblis 色盲模拟 / 对比工具](https://davidmathlogic.com/colorblind/) | 检查配色在各类色觉障碍下的可辨性 |
| [CARTO Colors](https://carto.com/carto-colors/) | 现代地图/信息图配色 |
| [Coolors](https://coolors.co/) | 快速生成/微调配色方案 |
