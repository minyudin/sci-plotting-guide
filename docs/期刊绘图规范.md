# 主流期刊绘图规范速查

> 以下为通用整理，投稿前务必以期刊官方 Guide for Authors 最新版为准。

## 通用要求（几乎所有 SCI 期刊适用）

| 项目 | 推荐值 |
|---|---|
| 单栏图宽 | 约 85–90 mm（3.3–3.5 英寸） |
| 双栏图宽 | 约 170–180 mm（6.7–7 英寸） |
| 分辨率（位图） | 照片 ≥300 dpi；线图/含文字图 ≥600–1200 dpi |
| 矢量格式 | PDF / EPS / SVG（线图、散点图、流程图首选） |
| 位图格式 | TIFF（无损）/ 高质量 PNG；避免低质量 JPEG |
| 字体 | Arial / Helvetica（无衬线），全文统一 |
| 最终字号 | 缩放到版面尺寸后 5–8 pt，最小不低于 5 pt |
| 线宽 | 最终版面 ≥0.25 pt，常用 0.5–1 pt |
| 子图标号 | a, b, c…（Nature 小写粗体）或 A, B, C…（Science/医学期刊常用） |

## Nature 系列

官方规范：https://www.nature.com/nature/for-authors/formatting-guide

- 图宽：单栏 89 mm，双栏 183 mm；最大高度约 247 mm（含图注）。
- 子图标号：**小写粗体** a、b、c，无括号。
- 首选矢量格式（AI/PDF/EPS）；文字用无衬线字体，最终字号 5–7 pt。
- 提倡色盲友好配色，避免仅用红/绿对比（Nature Methods 多次撰文推荐 Okabe-Ito）。

## Science（AAAS）

作者须知入口：https://www.science.org/content/page/science-information-authors

- 图宽：单栏约 5.5 cm（2.25 in），双栏约 12 cm（4.75 in），最宽 18.3 cm。
- 首次投稿可将图嵌入正文 PDF；修回/接收后需提供高分辨率单独文件。
- 建议 Helvetica/Arial；面板标号大写 A、B、C。

## Cell Press

- 图宽：1 栏 85 mm，1.5 栏 114 mm，2 栏 174 mm。
- 格式：优先 PDF/EPS/AI 矢量；位图 300–600 dpi TIFF。
- 面板标号大写 A、B、C。

## Elsevier 系列

官方规范：https://www.elsevier.com/about/policies-and-standards/author/artwork-and-media-instructions

- 图宽：单栏 90 mm，1.5 栏 140 mm，双栏 190 mm。
- 分辨率：半色调 300 dpi；线图 1000 dpi；混合图 500 dpi。
- 格式：EPS/PDF（矢量）、TIFF/JPEG（位图）；嵌入或转曲字体。

## PLOS

官方规范：https://journals.plos.org/plosone/s/figures

- 仅接受 **TIFF 或 EPS**。
- 宽度 6.68–19.05 cm，高度 ≤22.23 cm；300–600 dpi。
- 图注写在正文中，不放入图片文件内。

## IEEE

- 图宽：单栏 3.5 in（88.9 mm），双栏 7.16 in（181.8 mm）。
- 最终字号约 8–10 pt（与正文接近）；常用 Times/Helvetica。
- 提供 EPS/PDF/TIFF，位图 ≥300 dpi（线图 600 dpi）。

## 投稿前自查清单

- [ ] 图宽按目标期刊栏宽制作，未依赖投稿系统缩放
- [ ] 所有文字在最终版面尺寸下 ≥5 pt 且清晰可读
- [ ] 线图/流程图为矢量格式；位图分辨率达标
- [ ] 配色色盲友好，灰度打印仍可区分
- [ ] 各子图字体、字号、配色、坐标轴风格统一
- [ ] 坐标轴有物理量与单位；误差棒在图注中说明含义（SD/SEM/CI，n=?）
- [ ] 显微/电镜照片有比例尺；统计显著性标注方式在图注中说明
- [ ] 未做违规图像处理（拼接需标明分界，亮度/对比度调整须整图一致）
