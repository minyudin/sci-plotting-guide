## 3. 连续型 / 发散型科学色图（用于热图、等值线图等）

**原则：避免 jet/rainbow**，使用感知均匀（perceptually uniform）色图：

| 色图 | 类型 | 来源 |
|---|---|---|
| viridis / plasma / inferno / magma / cividis | 连续 | [Matplotlib 内置](https://matplotlib.org/stable/users/explain/colors/colormaps.html) |
| batlow / vik / roma / oleron 等 | 连续/发散/地形 | [Fabio Crameri 科学色图](https://www.fabiocrameri.ch/colourmaps/)（Nature Communications 推荐引用） |
| CET 系列 | 连续/发散/循环 | [colorcet](https://github.com/holoviz/colorcet) |
| RdBu / BrBG / PiYG 等 | 发散 | [ColorBrewer](https://colorbrewer2.org/)（可勾选 colorblind safe） |

## 4. 定性分类配色（地图、分组柱状图等）

- **ColorBrewer**：https://colorbrewer2.org/ — 经典之选，可筛选"色盲安全/打印友好"
- **CARTO Colors**：https://carto.com/carto-colors/ — 现代感强，适合地图和信息图
- **MetBrewer**：https://github.com/BlakeRMills/MetBrewer — 取自大都会博物馆藏画的艺术配色
- **R 配色大全**：https://github.com/EmilHvitfeldt/r-color-palettes — 收录数千个 R 配色包

## 5. 使用建议

1. 同一篇论文所有图使用同一套配色，同一变量/分组在不同图中颜色一致。
2. 分组 ≤ 8 时优先 Okabe-Ito；分组更多时考虑用形状/线型辅助区分。
3. 连续数据的色图必须配色标（colorbar），且发散色图的中心应对应有意义的零点。
4. 投稿前用灰度打印预览一次，确认去色后仍可区分。
